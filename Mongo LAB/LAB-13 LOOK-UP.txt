LAB-13 LOOK-UP

Part-A

1. Fetches each student and attaches matching department details using $lookup.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Department",
      	localField:"dept_id",
      	foreignField:"_id",
      	as:"department_info"
    }
  }
])

2. Joins students with departments and displays only the student’s name and their department name.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Department",
      	localField:"dept_id",
      	foreignField:"_id",
      	as:"department_info"
    }
  },
  {$unwind:"$department_info"},
  {
    $project:{
      _id:0,
      name:1,
      department:"$department_info.dept_name"
    }
  }
])

3. Retrieves every student along with their corresponding fee information.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Fees",
      	localField:"_id",
      	foreignField:"student_id",
      	as:"fee_info"
    }
  }
 
])

4. Finds students whose fee status is marked as “Unpaid”.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Fees",
      	localField:"_id",
      	foreignField:"student_id",
      	as:"fee_info"
    }
  },
  {
    $unwind:
      "$fee_info"
  },
  {$match:{"fee_info.status":"Unpaid"}}
 
])

5. Fetches students with both their department details and fee records in a single pipeline.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Fees",
      	localField:"_id",
      	foreignField:"student_id",
      	as:"fee_info"
    },
		 $lookup:{
        	from:"Department",
        	localField:"dept_id",
        	foreignField:"_id",
        	as:"dept_info"
      }
  }
])

6. Returns only those students for whom no matching department exists.

db.Student.aggregate([
  {
    $lookup: {
      from: "Department",
      localField: "dept_id",
      foreignField: "_id",
      as: "department"
    }
  },
  {
    $lookup: {
      from: "Fees",
      localField: "_id",
      foreignField: "student_id",
      as: "fees"
    }
  }
])

7. Selects students whose department name is "Computer Science" using join + match.

db.Student.aggregate([
  {
    $lookup: {
      from: "Department",
      localField: "dept_id",
      foreignField: "_id",
      as: "department"
    }
  },
  { $unwind: "$department" },
  { $match: { "department.dept_name": "Computer Science" } }
])

8. Performs a join with departments and sorts all students by their marks in descending order.
db.Student.aggregate([
  {
    $lookup: {
      from: "Department",
      localField: "dept_id",
      foreignField: "_id",
      as: "department"
    }
  },
  { $sort: { marks: -1 } }
])
9. Counts how many students belong to each department after joining.

db.Student.aggregate([
  {
    $lookup: {
      from: "Department",
      localField: "dept_id",
      foreignField: "_id",
      as: "department"
    }
  },
  { $unwind: "$department" },
  {
    $group: {
      _id: "$department.dept_name",
      totalStudents: { $sum: 1 }
    }
  }
])
