LAB-13 LOOK-UP

Part-A

1. Fetches each student and attaches matching department details using $lookup.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Department",
      	localField:"dept_id",
      	foreignField:"_id",
      	as:"department_info"
    }
  }
])

2. Joins students with departments and displays only the student’s name and their department name.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Department",
      	localField:"dept_id",
      	foreignField:"_id",
      	as:"department_info"
    }
  },
  {$unwind:"$department_info"},
  {
    $project:{
      _id:0,
      name:1,
      department:"$department_info.dept_name"
    }
  }
])

3. Retrieves every student along with their corresponding fee information.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Fees",
      	localField:"_id",
      	foreignField:"student_id",
      	as:"fee_info"
    }
  }
 
])

4. Finds students whose fee status is marked as “Unpaid”.

db.Student.aggregate([
  {
    $lookup:{
      	from:"Fees",
      	localField:"_id",
      	foreignField:"student_id",
      	as:"fee_info"
    }
  },
  {
    $unwind:
      "$fee_info"

])

Part – C 
1. Arranges all student records in descending order of marks.

])

2. Divides students into buckets based on marks: 0–60, 60–80, and 80–100. 
db.Student.aggregate([
  {
    $bucket: {
      groupBy: "$marks",
      boundaries: [0, 60, 80, 100],
      default: "Other",
      output: {
        count: { $sum: 1 },
        students: { $push: "$name" }
      }
    }
  }
])

5. Compares each student's marks with the average marks of their department
db.Student.aggregate([
  {
    $lookup: {
      from: "Department",
      localField: "dept_id",
      foreignField: "_id",
      as: "department"
    }
  },
  { $unwind: "$department" },
  {
    $group: {
      _id: "$department.dept_name",
      avgMarks: { $avg: "$marks" },
      students: {
        $push: {
          name: "$name",
          marks: "$marks"
        }
      }
    }
  },
  {
    $project: {
      _id: 1,
      avgMarks: 1,
      students: {
        $map: {
          input: "$students",
          as: "student",
          in: {
            name: "$$student.name",
            marks: "$$student.marks",
            aboveAverage: { $gte: ["$$student.marks", "$avgMarks"] }
          }
        }
      }
    }
  }
])
